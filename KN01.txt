1. Docker version überprüfen
	- docker --version
2. Docker search Befehle
	- docker search ubuntu
	- docker search nginx
3. Parameter Erklären
	- -d (startet den container im Hintergrund)
	- -p 80:80 (Mappt Port 80 des Hosts auf Port 80 des Containers)
	- docker/getting-started (Das verwendete Image)
4. Nginx Image 
	- docker pull nginx
	- docker create -p 8081:80 --name mynginx nginx
	- docker start mynginx
	- http://localhost:8081
5. Ubuntu Image
	- docker run -d ubuntu
Docker versuchr das Image zu finden. Wenn das Image nicht gefunden wird versucht der Docker das Image automatisch aus dem Docker Hub herunterzuladen. Der Docker wird im Hintergrund ausgeführt. Mit dem Befehl "Docker ps" kann ich jetzt schauen ob der docker gestartet ist.
	- docker run -it ubuntu
Mit dem "-it" Befehl öffnet man direkt das interactive terminal. Das interactive terminal ist in der shell im COntainer.
6. Nachträglich interactive shell starten
	- docker exec -it mynginx /bin/bash
	- service nginx status
7. status der docker
	- ps -a
8. NGINX stoppen
	- docker stop mynginx
9. Docker löschen
	- docker rm $(docker ps -a -q)
	- docker rm bf4d4c6c1b9b5c4420644e34a3240806732ed644dbd4c60555d7598ad5fa9c8a
	- docker rm b90345f2ec0fc37296b05ade9b4b1ec90defc96d75005795bd998748de07d244
	- docker rm 851af9f61af98fa4654d0b2966fb2d0b165d5fb892308cec0ab2c54e612d12f5
10. Images Entfernen
	- docker rmi nginx ubuntu
